# プライバシーポリシー
（施行日：2025年10月12日 / 最終改定日：2025年11月17日）

## 第1条（適用）
本プライバシーポリシー（以下「本ポリシー」といいます）は、当サービスを利用するすべてのユーザー（以下「ユーザー」といいます）に適用されます。

## 第2条（収集する情報）
当サービスでは、以下の情報を収集します。

### 2-1. ユーザー登録時に収集する情報
当サービスの一部機能を利用するためには、Google アカウントによるOAuth認証を通じたユーザー登録が必要です。ユーザー登録時に、以下の情報を収集します。

1. **Google アカウント情報**
   - 氏名（Googleアカウントの表示名）
   - メールアドレス
   - プロフィール画像のURL
   - Google ユーザーID

2. **Google OAuth認証情報**
   - アクセストークン
   - リフレッシュトークン
   - トークンの有効期限情報

これらの情報は、YouTube Data API との連携およびサービス提供のために使用されます。

### 2-2. すべてのユーザーから収集する情報

1. **アクセスログおよび利用状況情報**
   利用状況の把握や運営改善のため、IPアドレス、アクセス日時、閲覧ページ、検索キーワード、利用回数などの情報を記録する場合があります。
   これらの情報は統計的に処理され、個人を特定する目的では使用しません。

2. **クッキー（Cookie）**
   ユーザーの利便性向上、ログイン状態の維持、および利用規約への同意管理のため、クッキーを使用します。クッキーの利用は、ユーザーのブラウザ設定により制限または拒否することが可能です。
   クッキーを無効化した場合、一部の機能（ログイン状態の維持、利用規約への同意状態の保持など）が正しく動作しないことがあります。

## 第3条（情報の利用目的）
収集した情報は、以下の目的で利用します。

1. サービスの提供および運営
2. YouTube Data API との連携によるチャンネル情報の取得
3. ユーザー認証およびログイン状態の維持
4. サービスの改善および新機能の開発
5. サービスに関する通知やお知らせの送信
6. 利用状況の分析および統計データの作成
7. 不正利用の検知・防止

## 第4条（情報の管理および安全対策）
当サービスでは、収集した情報の適切な管理に努め、以下の安全管理措置を講じています。

1. **暗号化**
   - Google OAuth認証情報（アクセストークン、リフレッシュトークン）は暗号化して保存します
   - APIキーは暗号化して保存します
   - 通信は HTTPS により暗号化されます

2. **アクセス制限**
   - データベースへのアクセスは、必要最小限の管理者のみに制限されます

3. **ログの保持**
   - 不正利用の検知・防止のため、一定期間ログを保持します

4. **第三者への提供の制限**
   - 以下の場合を除き、第三者に提供することはありません
     - 法令に基づく開示要請があった場合
     - 公的機関からの正当な要請があった場合
     - ユーザーの同意がある場合

## 第5条（アカウント削除および情報の削除）
1. ユーザーは、プロフィール画面からいつでもアカウントを削除することができます。
2. アカウント削除時には、以下の情報が削除されます。
   - 登録されたユーザー情報（氏名、メールアドレス、プロフィール画像）
   - Google OAuth認証情報（アクセストークン、リフレッシュトークン）
   - ユーザーが登録したチャンネル情報
   - その他、ユーザーに関連するすべてのデータ
3. アカウント削除時、当サービスはGoogle側のOAuthトークンを無効化（revoke）します。これにより、当サービスがユーザーのGoogleアカウントにアクセスする権限が完全に削除されます。
4. 一度削除された情報は復元できません。
5. ただし、法令により保持が義務付けられている情報や、不正利用防止のために必要な最小限のログ情報については、一定期間保持する場合があります。

## 第6条（プライバシーポリシーの変更）
1. 本ポリシーは、必要に応じて変更することがあります。
2. 変更後の本ポリシーは、当サービス上に掲載された時点で効力を持つものとします。
3. 変更があった場合は、本サービス上に「変更内容」と「掲載日（発効日）」を明記して掲載します。
4. ただし、軽微な修正または誤記訂正等については、ユーザーへの個別通知を行わない場合があります。

## 第7条（お問い合わせ）
本ポリシーに関するお問い合わせは、当サービスのお問い合わせフォームまたはGitHubリポジトリのIssueよりご連絡ください。

## 第8条（準拠法・管轄）
本ポリシーの解釈および適用には、日本の法律が適用されます。
