# 利用規約
（施行日：2025年10月12日 / 最終改定日：2025年11月17日）

## 第1条（適用）
本規約は、「歌枠履歴er:D（うたわくりれきゃー:D）」（以下「当サービス」といいます）を利用するすべてのユーザー（以下「ユーザー」といいます）に適用されます。ユーザーは本規約に同意した上で当サービスを利用するものとします。

## 第2条（サービス内容）
1. 当サービスは、VTuberの歌配信に関するタイムスタンプを一覧表示、および検索するシステムを提供します。
2. 当サービスでは、管理者が任意のVTuberチャンネルを登録し、YouTube Data API を利用してアーカイブとコメントを取得し、一定のルールに基づきタイムスタンプ付きの配信をリスト化します。
3. ユーザーは、当サービスに登録されたアーカイブおよびタイムスタンプを閲覧することができます。
4. タイムスタンプにはYouTubeの該当箇所へのリンクが含まれており、ユーザーは直接該当の配信へ遷移することができます。
5. 当サービスでは、APIの呼び出し回数を抑制し、通信負荷を軽減する目的で、動画やコメントの情報を一時的に保存する場合があります。ただし、保存される情報には個人情報を含みません。
6. 保存された情報は適宜更新を行い、YouTube上の最新情報との整合性を維持するよう努めますが、必ずしも完全ではなく、YouTubeの情報と齟齬が発生する可能性があります。ユーザーは、最新の情報を確認する場合、YouTube上の公式情報を参照するものとします。
7. 検索機能は、保存されたデータを基に提供されますが、その網羅性や正確性を保証するものではありません。
8. 当サービスは非公式の有志プロジェクトであり、営利を目的として運営されるものではありません。

## 第3条（免責事項）
1. 当サービスは、YouTube Data API の提供する情報を基に構築されており、情報の正確性・完全性について保証するものではありません。
2. YouTubeの仕様変更やAPI利用制限により、サービスの一部または全部が予告なく変更・停止される場合があります。
3. 当サービスはYouTubeやVTuberに関する団体・個人とは一切関係がなく、公式な保証を受けたものではありません。
4. 検索結果に表示されない場合であっても、当該動画や情報が存在しないことを意味するものではありません。
5. 当サービスからリンクされている外部サイト（YouTubeを含む）の内容については、当サービスは一切の責任を負いません。
6. ユーザーは、必要に応じてYouTube上の公式情報を確認する責任を負うものとします。

## 第4条（禁止事項）
ユーザーは、当サービスを利用するにあたり、以下の行為をしてはなりません。
1. YouTube の利用規約やガイドラインに違反する行為
2. 不正アクセス、データの改ざん、その他の不正行為
3. 他者の権利（著作権・肖像権等）を侵害する行為
4. 当サービスのデータを自動的に取得・解析・再配布する行為（ただし、robots.txt 等で許可された一般的なクローリングやキャッシュは除きます）
5. 当サービスの検索結果を、第三者に誤解を与える形で利用・転載する行為
6. 当サービスの情報を、公式な情報源と誤認させる行為
7. 当サービスの運営を妨害する行為
8. その他、管理者が不適切と判断する行為

## 第5条（ユーザー登録および認証）
1. 当サービスの一部機能（チャンネル管理機能等）を利用するためには、ユーザー登録が必要です。
2. ユーザー登録は、Google アカウントによるOAuth認証を通じて行われます。
3. 当サービスは、ユーザー登録時に以下の情報を取得・保存します。
   - 氏名（Googleアカウントの表示名）
   - メールアドレス
   - プロフィール画像
   - Google OAuth認証に必要なトークン情報
4. 取得した情報は、サービスの提供およびYouTube Data API との連携に使用されます。詳細はプライバシーポリシーをご確認ください。
5. ユーザーは、取得した情報の取り扱いについて同意した上で、ユーザー登録を行うものとします。

## 第6条（アカウント削除）
1. ユーザーは、当サービスのプロフィール画面からいつでもアカウントを削除することができます。
2. アカウント削除時には、ユーザーに関連する以下の情報が削除されます。
   - 登録されたユーザー情報（氏名、メールアドレス、プロフィール画像）
   - Google OAuth認証に関するトークン情報
   - ユーザーが登録したチャンネル情報
3. アカウント削除時、当サービスはGoogle側のOAuthトークンを無効化（revoke）します。
4. 一度削除されたアカウント情報は復元できません。

## 第7条（クッキーの使用）
1. ユーザーは、初回アクセス時にクッキーの利用と本規約への同意を求められ、これに同意した上でサービスを利用するものとします。
2. 当サービスでは、以下の目的でクッキーを利用します。
   - ログイン状態の維持
   - 利便性向上
   - アクセス解析
3. ユーザーは、ブラウザの設定によりクッキーを無効化することができますが、一部の機能が正しく動作しない場合があります。

## 第8条（サービスの停止・終了）
1. 管理者は、以下の場合において当サービスの全部または一部を事前の通知なく停止・終了することができます。
   - システムの保守・更新を行う場合
   - YouTube Data API の利用制限や仕様変更によりサービスの継続が困難になった場合
   - その他、管理者がサービスの継続を困難と判断した場合
2. 検索機能を含むサービス内容は、事前の通知なく変更・停止される場合があります。
3. サービスの停止・終了により、ユーザーに損害が生じた場合であっても、管理者は一切の責任を負わないものとします。

## 第9条（規約の変更）
1. 管理者は、必要に応じて本規約を変更することができるものとします。変更後の規約は、当サービス上に掲載された時点で効力を持つものとします。
2. ただし、軽微な修正または誤字訂正等については、ユーザーへの個別の通知を行わない場合があります。

## 第10条（準拠法・管轄）
本規約の解釈および適用には、日本の法律が適用されます。
